Descrição do caso escolhido
A TAKAY BRECHÓ é uma loja de roupas de brechó que oferece peças exclusivas, estilosas e acessíveis. O projeto foi desenvolvido para criar uma presença digital para a loja, facilitando a divulgação dos produtos e o contato com os clientes.
Necessidade identificada
Foi identificado que a loja não possuía um site próprio para apresentar seus produtos e divulgar sua marca na internet. A criação do site busca melhorar a visibilidade da loja, facilitar o acesso às informações e oferecer uma experiência mais organizada para os clientes.
Público-alvo
O público-alvo é composto por jovens e adultos interessados em moda sustentável, roupas exclusivas, peças vintage e produtos de qualidade com preços acessíveis.
Objetivo do site
O principal objetivo do site é apresentar os produtos da TAKAY BRECHÓ, divulgar a marca e facilitar o contato dos clientes por meio do Instagram e WhatsApp. Além disso, o site permite que o usuário simule uma compra utilizando o carrinho de produtos.
Processo de desenvolvimento
O desenvolvimento do projeto ocorreu em etapas:
1.	Escolha do negócio real para o estudo de caso.
2.	Levantamento das necessidades da loja.

4.	Desenvolvimento da estrutura HTML.
5.	Criação da identidade visual utilizando CSS.

9.	Preparação para publicação do projeto.
Principais decisões do projeto
Estrutura HTML
Foi utilizada uma estrutura semântica contendo:
•	Header
•	Nav
•	Section
•	Article
•	Footer
Essa organização facilita a leitura do código e melhora a acessibilidade.
Organização visual
Foram utilizadas cores preto, branco e vermelho para representar a identidade visual da marca, inspirada no estilo japonês da TAKAY BRECHÓ.
Responsividade
Foi utilizado Bootstrap e recursos CSS para garantir boa visualização em computadores, tablets e celulares.
Acessibilidade
Foram adicionados textos alternativos (alt) nas imagens e uma organização hierárquica dos títulos para facilitar a navegação.
Interatividade
Foi implementado um carrinho de compras simples para permitir a seleção dos produtos.
Funcionalidade em JavaScript
A funcionalidade principal desenvolvida em JavaScript é o carrinho de compras.
O sistema permite:
•	Adicionar produtos ao carrinho.
•	Atualizar a quantidade de itens.
•	Atualizar o valor total da compra.
•	Exibir um resumo da compra ao finalizar o pedido.
•	Limpar o carrinho após a finalização.
Essa funcionalidade foi escolhida por estar diretamente relacionada ao objetivo do site.
Uso de Bootstrap
Foram utilizados os seguintes recursos do Bootstrap:
•	Navbar responsiva.
•	Sistema de Grid.
•	Containers.
•	Botões estilizados.
•	Classes utilitárias de espaçamento.
•	Responsividade automática para diferentes dispositivos.
O Bootstrap foi utilizado para facilitar a adaptação da página em diferentes tamanhos de tela e melhorar a aparência geral do projeto.
